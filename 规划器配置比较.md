# 局部规划器配置比较

## 配置文件与规划器对应关系

| 配置文件 | 控制器插件 | 局部规划器类型 | 规划器插件 |
|---------|-----------|--------------|-----------|
| **nav2_params.yaml**（原始） | `["FollowPath"]` | DWB | `dwb_core::DWBLocalPlanner` |
| **nav2_params_teb.yaml** | `["FollowPath"]` | TEB | `teb_local_planner::TebLocalPlannerROS` |
| **nav2_params_intpc.yaml** | `["IntpcController"]` | IntPC | `intpc_local_planner::IntpcLocalPlannerROS` |

## 详细配置说明

### 1. 原始配置文件 (nav2_params.yaml)
```yaml
controller_plugins: ["FollowPath"]

# DWB规划器配置
FollowPath:
  plugin: "dwb_core::DWBLocalPlanner"
  # 其他DWB特有的参数...
```

### 2. TEB配置文件 (nav2_params_teb.yaml)
```yaml
controller_plugins: ["FollowPath"]  # 仅包含TEB规划器

# TEB规划器配置
FollowPath:
  plugin: teb_local_planner::TebLocalPlannerROS
  # 其他TEB特有的参数...
```

### 3. IntPC配置文件 (nav2_params_intpc.yaml)
```yaml
controller_plugins: ["IntpcController"]  # 仅包含Intpc规划器

# Intpc规划器配置
IntpcController:
  plugin: intpc_local_planner::IntpcLocalPlannerROS
  # 其他IntPC特有的参数...
```

## 总结

- **原始配置文件**（nav2_params.yaml）默认使用的是 **DWB规划器**
- **TEB配置文件**（nav2_params_teb.yaml）专门使用 **TEB规划器**
- **IntPC配置文件**（nav2_params_intpc.yaml）专门使用 **IntPC规划器**

这三个配置文件是为了支持不同的局部规划算法而分别设计的，您可以根据需要选择使用其中的一个配置文件来启动导航系统。
